# File Exchange

A lightweight LAN-based file transfer app built with Python and PyQt5.
Discover devices on your local network and send files directly between
them — no internet, cloud, or account required.

## Features
- Automatic device discovery on the local network
- Drag-and-drop file sending, including multiple files at once
- Real-time progress tracking
- Files saved directly to your Downloads folder
- Packaged as a standalone Windows executable
